A small library project to look for books and save the one you want to read or buy later on

To run the project locally all you need to do is

1. Start by cloning this repository.
2. Navigate to the project folder and run npm install.
3. Obtain your own API KEY from Google Cloud.
4. Once you have your key, you can either:

Paste it as a string directly in the api.tsx file to the apiKEY variable, or
Create a .env file and define a VITE_ variable (e.g., VITE_API_KEY=your_key_here).
and use in the api.tsx file on the apiKEY variable.
